# AppMeteo
Application de météo à l'aide de l'Api OpenWeather permettant de me familiariser avec React.

1- Télécharger le projet
2- Lancer le projet dans un environnement qui supporte REACT (style visual studio code)
3- Ouvrir le terminal et lancer la commande 'npm install' (assurez vous d'avoir la possibilité de lancer ce genre de commande sur votre machine)
4- Lancer l'application à la fin de l'installation avec un 'npm run dev' et en cliquant sur le lien localhost qui s'affiche dans le terminal
